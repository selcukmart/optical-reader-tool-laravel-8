###### Bu Sınıf Ne İşe Yarar?

Tüm para birimlerini select box şeklinde sunar 

###### Nasıl Kullanılır

```
             [
                'type' => 'currency',
                'label' => 'Para Birimi',
                'attributes' => [
                    'name' => 'currency'
                ]
             ]
```

Tüm para birimlerini gösterir.

###### Hangi Durumlarda Kullanılır

Para birimine ihtiyaç duyulduğunda kullanılır.
