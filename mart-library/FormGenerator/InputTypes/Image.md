###### Bu Sınıf Ne İşe Yarar?

Kapak resmi eklemek için kullanılır. ...


###### Nasıl Kullanılır

```
[
                'type' => 'image',
                'label' => 'Kapak Resmi Yükle',
                'attributes' => [
                    'name' => 'foto'
                ]
            ],
```

Type kısmında sadece image olduğu belirtilir

###### Hangi Durumlarda Kullanılır
Resim ekleme ihtiyacı duyulan yerlerde eklenir.
