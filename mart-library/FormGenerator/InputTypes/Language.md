###### Bu Sınıf Ne İşe Yarar?

Ülkelerin dilimlerini select box şeklinde sunar

###### Nasıl Kullanılır

```
                [
                'type' => 'language',
                'label' => 'Dil',
                'attributes' => [
                    'name' => 'code'
                ]
                  ]
```

Tüm ülkelerin dillerini gösterir.

###### Hangi Durumlarda Kullanılır

Bir ülkenin dil seçeneğine ihtiyaç duyulduğunda kullanılır.
